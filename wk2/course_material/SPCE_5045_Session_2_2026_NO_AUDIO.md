# Aspects of “Operability”

<span style="color:#000000">1\. </span>  <span style="color:#000000"> __visibility/observability__ </span>  <span style="color:#000000">: the extent to which the system provides the operations team with usable information about the configuration\, status\, and performance of the system\. </span>

<span style="color:#000000">2\. </span>  <span style="color:#000000"> __commandability/controllability__ </span>  <span style="color:#000000">: the extent to which the operations team can place the flight system in the desired state and produce the desired outcome via commanding\. </span>

<span style="color:#000000">3\. </span>  <span style="color:#000000"> __predictability__ </span>  <span style="color:#000000">: the extent to which the operations team is able to predict\, with some certainty\, the outcome of the execution of a planned event\. </span>

<span style="color:#000000">4\. </span>  <span style="color:#000000"> __flexibility__ </span>  <span style="color:#000000">: the extent to which the operations team can reconfigure components to maximize or optimize component utilization\, to circumvent anomalous components\, provide options\, to increase robustness\. </span>

<span style="color:#000000">5\. </span>  <span style="color:#000000"> __robustness__ </span>  <span style="color:#000000">: the extent to which the system maintains performance under perturbations and prevents and contains errors\. </span>

<span style="color:#000000">6\. </span>  <span style="color:#000000"> __autonomy__ </span>  <span style="color:#000000">: the extent to which the system manages nominal or contingency operations without ground intervention\. </span>

<span style="color:#000000">7\.</span>  <span style="color:#000000"> __ efficiency__ </span>  <span style="color:#000000">: the extent to which the operations team can optimize the use of time and resources\. </span>

<span style="color:#000000">8\. </span>  <span style="color:#000000"> __testability__ </span>  <span style="color:#000000">: the extent to which the operations team can verify and validate system components and test assets\. </span>

<span style="color:#000000">9\. </span>  <span style="color:#000000">tractability</span>  <span style="color:#000000">: the extent to which the operations team is freed from the need to pay attention to\, or “care and feed” the system\. </span>
